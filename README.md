# MTG Price Predictor
A Python-based Magic the Gathering card price predictor.

## Dev Setup
Create a venv and activate it:
```sh
python -m venv venv

# Windows (powershell)
.\venv\Scripts\Activate

# MacOS/Linux
./venv/Scripts/activate
```

Setup the dependencies:
```sh
pip install -r requirements.txt
```

> [!IMPORTANT]
> Please, always keep pip dependencies updated using `pip freeze > requirements.txt`
